# 2023 THE F2E Mission 2 - 總統即時開票全台地圖

## 作品說明

總統開票地圖，可讓使用者透過圖表了解各地開票資訊

## Designer

## 系統說明

- Node 版本: `v18.14.1`
- pnpm 版本: `v8.2.0`

- 安裝： `pnpm install`
- 執行： `pnpm start`

### 資料夾說明

```

 app
    | --- components (共用元件、頁面子元件)
    | --- configs     (常數變數)
    | --- models   (定義資料interface)
    | --- pages    (頁面元件)
    | --- services (api函式)
    | --- shell    (layout)
    | --- style    (共用樣式、顏色變數定義）
    | --- store    (全域資料)
    | --- utils    (共用函式)
    | ---

```

### 使用技術

- Angular v17
- Rx js
- D3
- Chart.js
